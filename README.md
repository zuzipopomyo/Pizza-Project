# 🍕 Pizza Project

A modern pizza ordering frontend built with Next.js and Tailwind CSS.

## ✨ Features

- Responsive pizza UI
- Product cards
- Cart functionality
- Modern animations
- Mobile-friendly design
- Fast Next.js performance

## 🛠 Tech Stack

- Next.js
- React
- Tailwind CSS
- JavaScript / TypeScript

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/zuzipopomyo/Pizza-Project.git
```

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open:

```txt
http://localhost:3000
```
